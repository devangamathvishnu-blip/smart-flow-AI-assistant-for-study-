SmartFlow AI Assistant for Study

Project Overview

SmartFlow AI Assistant for Study is an AI-powered educational assistant developed using n8n workflow automation and OpenAI ChatGPT. The assistant helps students by answering academic questions, generating study notes, providing explanations, and maintaining conversational memory for a better learning experience.

Features

- AI-powered question answering
- Conversational memory support
- Study assistance and learning guidance
- Structured educational responses
- Integration with OpenAI ChatGPT
- n8n workflow automation
- Google Sheets integration

Technologies Used

- n8n
- OpenAI ChatGPT API
- Simple Memory
- Google Sheets
- GitHub

Workflow Architecture

User Query
↓

↓

↓
OpenAI Chat Model
↓
Memory Module
↓

↓
Response Generation

Applications

- Student Learning Support
- Academic Query Resolution
- Exam Preparation
- Study Note Generation
- Educational Assistance

Advantages

- Improves learning efficiency
- Available 24/7
- Easy to use
- Personalized interaction through memory
